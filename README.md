# 👨‍💼 HR Analytics Dashboard (Excel Project)

## 📌 Project Overview

This project is an **Excel-based HR Analytics Dashboard** developed to analyze employee data and provide meaningful insights into workforce demographics, employee performance, salary distribution, and organizational trends. The dashboard transforms raw HR data into interactive visualizations that help support data-driven HR decision-making.

The project demonstrates practical skills in:

- Data Cleaning
- Data Preprocessing
- Data Analysis
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Business Insight Generation
- Data Visualization in Excel

---

# 📊 Dataset Information

The dataset used for this project is **HRDataset_v14-selected-columns.csv** and contains employee-related information.

| Column Name | Description |
|--------------|-------------|
| Employee_Name | Name of the employee |
| EmpID | Unique employee identifier |
| MarriedID | Indicates whether the employee is married |
| MaritalStatusID | Employee marital status ID |
| GenderID | Gender identifier |
| EmpStatusID | Employment status identifier |
| DeptID | Department identifier |
| PerfScoreID | Performance score identifier |
| FromDiversityJobFairID | Indicates if employee was hired through a Diversity Job Fair |
| Salary | Annual employee salary |
